# Shader Enhancement Plan

## Enhancements to Existing Shaders:
1. **Plasma Vibrant (ID 1)** - Add multi-layer FBM noise, more complex color blending
2. **Tunnel Neon (ID 2)** - Add hexagonal patterns, chromatic aberration, depth fog
3. **Raymarcher SDF (ID 3)** - Add multiple shapes (boxes, torus), lighting, shadows
4. **Wave Distortion (ID 6)** - Add complex feedback loops, chromatic splits
5. **Particle System (ID 7)** - Add particle trails, better physics, variety

## New Spectacular Shaders to Add:
